# Media Fetcher

A simple web app that allows users to fetch publicly available video content using a video URL.

Built with:

- ⚙️ FastAPI backend  
- ⚛️ Next.js frontend (TypeScript)  
- 🎥 yt-dlp for media extraction (supports many platforms)
